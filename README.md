\# Runner 3D Game



A simple 3D runner game built with Unity for my \*\*MAC280 Game Programming\*\* course.



\---



\## Overview



\*\*Runner 3D Game\*\* is a Unity project where the player controls a character in a 3D environment and avoids obstacles while moving forward.



The project focuses on basic game programming concepts such as:



\- Player movement

\- Obstacle interaction

\- Scene setup

\- Basic gameplay flow

\- Unity project structure



\---



\## Features



| Feature | Description |

|---|---|

| 3D Gameplay | Player moves through a 3D runner-style environment |

| Player Movement | Basic movement controls for the main character |

| Obstacles | Obstacles are placed in the scene for gameplay challenge |

| Unity Scene | Main game scene created inside Unity |

| Course Project | Built as part of MAC280 Game Programming |

\## Tech Stack
